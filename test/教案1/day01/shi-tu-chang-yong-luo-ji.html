字典转换JSON

from flask import Flask, json
app = Flask(__name__)
@app.route('/')
def index():
    json_dic = {
        "name": "laowang",
        "age": 18
    }
    # json.dumps 引用json 通过dumps将字典转换成字符串 result_json的类型是字符串，而浏览器获取到的就是字符串
    result_json = json.dumps(json_dic)  
    return result_json 

app.run()

JSON字符串转换成字典

from flask import Flask, json

app = Flask(__name__)

@app.route('/')
def index():  
    # 通过json.loads 将json字符串转换成字典
    json_loads = json.loads('{"age": 18, "name": "laowang"}')
    print(type(json_loads))
    return 'json'

app.run()

返回JSON

在使用 Flask 写一个接口时候需要给客户端返回 JSON 数据，在 Flask 中可以直接使用 jsonify 生成一个 JSON 的响应

# 返回JSON
@app.route('/demo4')
def demo4():
    json_dict = {
        "user_id": 10,
        "user_name": "laowang"
    }
    return jsonify(json_dict)

不推荐使用 json.dumps 转成 JSON 字符串直接返回，因为返回的数据要符合 HTTP 协议规范，如果是 JSON 需要指定 content-type:application/json

重定向

  • 重定向到 黑马 官网
# 重定向
@app.route('/demo5')
def demo5():
    return redirect('http://www.itheima.com')
  • 重定向到自己写的视图函数
    • 可以直接填写自己 url 路径
    • 也可以使用 url_for 生成指定视图函数所对应的 url
@app.route('/demo1')
def demo1():
    return 'demo1'

# 重定向
@app.route('/demo5')
def demo5():
    return redirect(url_for('demo1'))
  • 重定向到带有参数的视图函数
    • 在 url_for 函数中传入参数
# 路由传递参数
@app.route('/user/<int:user_id>')
def user_info(user_id):
    return 'hello %d' % user_id

# 重定向
@app.route('/demo5')
def demo5():
    # 使用 url_for 生成指定视图函数所对应的 url
    return redirect(url_for('user_info', user_id=100))

自定义状态码

  • 在 Flask 中，可以很方便的返回自定义状态码，以实现不符合 http 协议的状态码，例如：status code: 666
@app.route('/demo6')
def demo6():
    return '状态码为 666', 666
